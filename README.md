# ⚡ PikaDash: Volt Rush

## 📝 Descripción del Proyecto
Un dinámico juego arcade en 2D desarrollado en C++ con SFML. Controla los reflejos de Pikachu con la barra espaciadora para esquivar obstáculos, usar power-ups y lograr la máxima puntuación antes de perder tus 3 vidas.

## 🎯 Objetivo del Juego
El objetivo principal es sobrevivir el mayor tiempo posible dentro de los límites del mapa, esquivando obstáculos y recolectando power-ups para acumular la mayor cantidad de puntos antes de que se agoten las 3 vidas del jugador.

## 🎮 Controles
* **Espacio (Space):** Cambiar instantáneamente la dirección de Pikachu.

## ⚙️ Mecánicas
* **Movimiento Automático:** El personaje corre sin parar, el jugador solo controla los cambios de dirección.
* **Sistema de Vidas:** El jugador inicia con 3 vidas, las cuales se pierden al chocar con obstáculos o salir del mapa.
* **Power-up de Invencibilidad:** Al recoger un ítem especial, Pikachu se vuelve invulnerable temporalmente y gana un multiplicador de puntos.

## 🏆 Características
* Estética retro 8-bits (Pixel Art).
* Sistema de colisiones preciso.
* Interfaz de usuario con puntuación y contador de vidas.
* Acción rápida y basada en reflejos.
